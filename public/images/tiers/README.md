# Tier Character Images

วางรูปตัวการ์ตูน Dreame mascot สำหรับแต่ละ tier ที่นี่ ขนาดแนะนำ:
- Aspect ratio: **3:4** (เช่น 600 × 800 px)
- Format: PNG หรือ WebP (transparent หรือ gradient bg)
- ขนาดไฟล์: < 300 KB ต่อรูป

## ไฟล์ที่ต้องมี

| File | Tier | Points Range | Color theme |
|---|---|---|---|
| `plus.png`   | PLUS   | 0 – 500       | Silver / gray gradient |
| `pro.png`    | PRO    | 501 – 1,500   | Black / dark gradient |
| `ultra.png`  | ULTRA  | 1,501 – 3,500 | Orange / black gradient |
| `master.png` | MASTER | 3,500+        | Iridescent / gold gradient |

## Fallback

ถ้าไม่มีไฟล์ระบบจะ render geometric placeholder ที่มี gradient พื้นหลังเลือกให้แล้ว
แทนที่ไฟล์ได้ทุกเมื่อ — ระบบจะตรวจอัตโนมัติ
