# VIP Treatment Benefit Icons

วางรูปไอคอน 3D / mascot สำหรับการ์ด "Your VIP Treatments" ในหน้า home

## ขนาดแนะนำ
- **Square 1:1** (เช่น 400 × 400 px หรือ 600 × 600 px)
- Format: **PNG transparent** (พื้นหลังโปร่ง — เพื่อให้ blend กับ white card)
- ขนาดไฟล์: < 200 KB ต่อรูป

## ไฟล์ที่ต้องมี

| ไฟล์ | ใช้กับการ์ด |
|---|---|
| `member-pricing.png`   | Member-Only Pricing  |
| `member-events.png`    | Special Member Events |
| `points.png`           | Points with Every Purchase |
| `exclusive-treats.png` | Exclusive Treats |

## Fallback
ถ้าไม่มีไฟล์ ระบบจะแสดงเป็นกรอบว่างพื้นหลังขาว (ไม่มี icon)
ใส่รูปทันทีเมื่อพร้อม — ระบบจะ render ทันทีโดยไม่ต้อง deploy ใหม่

## หมายเหตุ
- ใช้รูป 3D character / icon style แบบ Dreame website
- พื้นหลัง transparent เพื่อให้รูปดูลอย ไม่มีขอบ
- ถ้าต้องการอ้างอิง ดู: https://www.dreametech.com/pages/dreame-membership
